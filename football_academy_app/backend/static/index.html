Welcome to the Football Academy API

This API provides endpoints for managing player profiles, training plans, challenges, achievements, and player statistics. It allows players to track their progress, complete challenges, and improve their football skills.

View API Docs (Swagger UI) View API Docs (ReDoc)

API Overview

The API is organized into the following categories:

  • Users: User registration, profile management
  • Training Plans: Manage training plans and activities
  • Exercises: Browse and access exercise instructions
  • Achievements: View and manage player achievements and badges
  • Player Stats: Track player performance statistics
  • Challenge Progress: Record and view challenge completions

Getting Started

To start using the API:

  1. Register a new user account or use the test credentials
  2. Obtain an authentication token using the /token endpoint
  3. Use the token in the Authorization header for subsequent requests
  4. Explore the API endpoints using the documentation
